Аксессоры «get» и «set»

С помощью аксессоров можно задавать значения или их получать

Объявление аксессоров

        public int weight;

        public int Weight{
            get{
                return this.weight;
            }
            set{
                this.weight = value; (параметр по умолчанию)
            }
        }

Обращение к акссесору

        Class bot = new Class();
        bot.Weight = -100;
        Console.WriteLine(bot.Weight);

Отдельный акссесор:

        public int Width{ get; set;}

Таким методом можно заприщать и разрешать ввод или получение данных

Пример:

        public int Width{ private get; set;}
