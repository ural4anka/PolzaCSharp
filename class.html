Создание классов и объектов

Классы можно создавать, как в одном документе, так и в отдельном файле

Создание класса в новом документе: Проект/Добавление класса.../ Выбираем тип файла и нажимаем на кномпку "добавить"

Структура файла:

    using System;
    using System.Collections.Generic;
    using System.Text;

    namespace ConsoleApp3
    {
        class Robot
        {

        }
    }

Модификаторы доступа:

  • public - доступ везде

  • protected - доступ исключительно в классе объявления и в классах наследования

  • private - доступ исключительно в классе объявления

Дополнительные модификаторы доступа:

  • Добавить в дальнейшем...

Создание полей

модификатор_доступа тип_поля название_поля = значение_поля;

Пример:

        public int id;
        public string name;
        public int weight;
        public byte[] coordinates;

Полем называется переменная, которая создана внутри класса, а переменная вне класса

Создание ОБЪЕКТА

название_класса название_объекта = выделение_памяти_для_объекта();

Пример:

        Robot bot = new Robot();
        bot.id = 0;
        bot.name = "Bot";
        bot.weight = 800;
        bot.coordinates = new byte[] {0,0,0};

        Console.WriteLine(bot.name + "weight" + bot.weight);

Объект создается внутри основного класса

Работа с модификатором доступа private | protected

Чтобы объект обратился к полю с доступом private | protected, нужно использовать МЕТОДЫ

Создание методов:

        модификатор_доступ ключевое_слово(можно не использовать) тип_возвращения(если метод не будет ничего возврачать, то пишем void) название(тип_параметра нахвание_параметра (принимаемые параметры)){
            действие
        }

Пример:

        class Robot{
            private string name;
            private int weight;
            private byte[] coordinates;

            public void setValue(string _name, int _weight, byte[] _coordinates){
                name = _name;
                weight = _weight;
                coordinates = _coordinates;
            }

            public void printValue(){
                Console.WriteLine(name + "weight" + weight + ", coordinates:");
                foreach(byte el in coordinates){
                    Console.WriteLine(el)
                }
            }
        }
        class Main(){
            Robot bot = new Robot();
            bot.setValue("Bot", 800, new byte[] {0, 0, 0})
            bot.printValue();
        }
