Работа с файлами и строками

Создание строк

string word = "Hello";

Работа со строкой

word += "!!!";

Любая строка - массив из множества символов

char[] sym = {'s','o','m','e'};

Вывод символа e строки word:

Console.WriteLine(word[1]);

Вывод кол-ва символов строки:

Console.WriteLine(word.Length);

Функции для работы со строкой

Добавление символов к строке:

    word = String.Convat(word, "!!");
    Console.WriteLine(word);

Сравнивание строк

    Console.WriteLine(String.Convat(word, "Hello"));

Разделение на строки:

    string people = "Alex,Bob,John";
    string[] names = people.Splir(new char[]{','});
    foreach(string el in names){
        Console.WriteLine(el);
    }

Преобразование массива в строку:

    string people = "Alex,Bob,John";
    string[] names = people.Splir(new char[]{','});
    string name = String.Join(" ", names);
    Console.WriteLine(name);

Удоление лишних пробелов до или после строки:

    string word = "      Hello";
    Console.WriteLine(word.Trim());

Обрезание строки

    Console.WriteLine(word.Substring(0, word.Length - 1));

Работа с файлами

Чтобы работать с файлами нужно подключить билиотеку IO

using System.IO

Внутри основного метода нужно подключить директиву using

    using(){

    }

using служит для создания изолированного кода

Объявление для внесения данных:

    принимаем данных с консоли или с переменной
    using(тип_операции название_операции = выделяем_память_под_операцию("название_файла", перечисление.режим_открытия)){
        переводим текст в байты:
        1. создаем массив байтов
        byte[] array = Text.Encoding.Defualt.GetBytes(переменная_которую_переводим_в_байты);
        2. записываем массив байтов
        stream.Write(array, отступ 0, до какого элемента будет записанно array.Length);
    }

Объявление для считывания данных:

    using(тип_операции название_операции = обращение_к_глабальному_классу_File.OpenRead("название_файла")){
        1. создаем массив байтов
        byte[] array = new byte[создаем длину массива];
        2. записываем массив байтов
        stream.Read(куда записываем array, отступ в начале 0, до какого элемента будет записанно array.Length);
        3. Создаем переменную куда информация из файла будет записываться
        string textFromFile = Text.Encoding.Defualt.GetString(переменная_которую_переводим_в_строку);
        4. Выполняем любое действие с этой записью
        Console.WriteLine(textFromFile);
    }

Режимы открытия

  1. Append - открыть файл и добавить информацию в него

  2. Create - открыть файл для его создания

  3. CreateNew - открыть файл для создания, если такого файла не существует

  4. Open - открыте файла, если такого файла несуществует, то будет вызываться ошибка

  5. OpenOrCreate - открыте файла, если такого файла несуществует, то будем его создавать

  6. Truncate - открыте файла для его полной очистки или удаления

Внесение данных:

    Console.WriteLine("Введите текст: ");
    string text = Console.ReadLine();
    using(FileStream stream = new FileStream("info.txt", FileMode.OpenOrCreate))){
        byte[] array = Text.Encoding.Defualt.GetBytes(text);
        stream.Write(array, 0, array.Length);
    }
