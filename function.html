Функции | Методы

Исходные (входные) - это данные, известные перед выполнением задачи из условия.

Промежуточные - данные, которые не являются ни аргументом, ни результатом алгоритма, а используются только для обозначение промежуточного значения.

Выходные - результат решения задачи.

Область видимости переменной - место в программе, в которой доступно значение переменной.

  • Локальные переменные доступны конкретной подпрограмме.
  • Глобальные переменные доступны всей программе.

Метод - это часть программы, которая может вызываться в коде основной программы и производить какие-либо действия с переданными в нее переданными.

Функция находится вне класса, а метод он внутри класса.

Создание метода, осуществляется внутри класса и не внутри другого метода.

Методу свойственно явное возвращение результата его работы.

Объявление метода:

        модификатор_доступа возвращаемый_тип название_метода(принимаемый_тип_данных название_типа_данных){
            действие
        }

Пример:

        public static void Print(string word){
            Console.WriteLine(word);
        }

Обращение к методу:

        static void Main(){
            Print("Hello!");
        }

Возврощение значения

        static void Main(string[] args)
        {
            int a = 4;
            int b = 5;
            int res = Summa(a, b);
            Print("Результат: " + res.ToString());
        }
        public static int Summa(int a, int b)
        {
            return a + b;
        }
        public static void Print(string word)
        {
            Console.WriteLine(word);
        }
