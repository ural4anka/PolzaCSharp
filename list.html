Работа с классом List | Динамические массивы

Сначало нужно подключить библиотеку Generic

        using System.Collections.Generic;

Реализация деннамического списка | массива

        List<тип данных> название_списка = new List<тип данных>(){
            значения
        };

Пример:

        List numbers = new List(){
            4, 6, 7
        };

Вывод динамического массива

        foreach(int el in numbers){
            Console.WriteLine("EL:" + el);
        }

Методы работы с классом List

Добавление:

numbers.Add(40);

Удаление:

numbers.Remove(40);

Сортировка массива:

numbers.Sort();

Реверс(перевернуть):

numbers.Reverse();

Очистка массива:

numbers.Clear();
