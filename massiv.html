Одномерный массив

Массив предоставляет собой совокупность переменных одного типа с обшим для обращения к ним именем.

Объявление массива:

        тип[] имя_массива = new тип[размер] {значение}

Пример:

        int[] array = new int[5] {1,2,3,4,5}

Разновидности объявлений массива:

        int[] array = new int[]
        int[] array = {1,2,3,4,5}
        int[] array = new int[5]
        int[] array = new int[5] {1,2,3,4,5}

Перебор массива через цикл for

        int[] array = new int[5] {1,2,3,4,5}
        for(int i = 0; i < 5); i++) {
            Console.WriteLine($"array[{i}] = {array[i]}")
        }

название_массива.lenght - длина массива

Двухмерный массив | Матрица

Матрица - это массив, в котором каждый элемент имеет два индекса (номер строки и номер столбца).

Объявление двухмерного массива:

        int[,] A = new int[4,3] {
            {-9, -6, -3},
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };

Описание массива:

        Random r = new Random()
        for(int i=0; i < 4; i++){
            for(int j=0; i < 3; i++){
                A[i,j]=r.Next(0,10);
            }
        }

Вывод матрицы:

        for(int i=0; i < 4; i++){
            for(int j=0; i < 3; i++){
                if(j!=3-1){
                    Console.Write(A[i,j] + "\t");
                }
                else{
                    Console.Write(A[i,j]);
                }
            }
            Console.WriteLine();
        }
