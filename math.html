Матиматические вычесления

Math.Abs(x) Модуль
Math.Ceiling(x) Округление числа до бльшего целового
Math.Floor(x) Округление числа до меньшего целового
Math.Cos(x) Косинус аргумента x
Math.Sin(x) Синус аргумента x
Math.Tan(x) Тангенс аргумента x
Math.Sqrt(x) Квадратный корень
Math.E Число e = 2,718282...
Math.Exp(x) Экспонента, число e в степени x
Math.Log(x) Логарифм числа x
Math.Log10(x) Логарифм десятичный числа x
Math.Max(x,y) Максимальное число из двух чисел
Math.Min(x,y) Минимальное число из двух чисел
Math.Pow(x,y) Число x в степени y
Math.Pi Число Пи
Math.Round(x) Простое округление числа
Math.Sign(x) Знак числа x
