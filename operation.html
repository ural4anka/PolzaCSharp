Операторы

Условный оператор If Else

        if(условие){
            действие
        }
        if else(условие){
            действие
        }
        else(условие){
            действие
        }

Оператор множественного выбора SWITCH

            switch (переключатель) {
                case значение 1:
                    инструкция 1;
                    break;
                case значение 2:
                    инструкция 2;
                    break;
                ...
                defualt:
                    инструкция_по_умолчанию;
                    break;
            }

break - выход из определенного условия

Циклические операторы

Цикл - многократное повторение одного и того же участка программы, пока заданное условие не станет ложным.

Существует два типа циклов: типа "пока" и типа "n-раз".

Первый тип "пока" предназначен для повторения некоторых действий до тех по, пока выполняется некоторое условие.

Второй тип "n-раз" предназначен для повторений некоторых действий зарание известное кол-во раз.

FOR

Оператор for содержит три параментра

1. Название инициализации

2. Условие повторения

3. Интерацию

        for(инициализация; условие; интерация){
            тело цикла
        }

Пример:

        for(int i = 0; i <=10; i++){
            Console.WriteLine(i);
        }

инициализация - объявление и описание переменной

WHILE

            while(условие_истинно){
                операторы
            }

Пример:

            int i = 1;
            while(i <= 10){
                Console.WriteLine(i);
                i++
            }

DO WHILE

                do{
                    операторы
                }while(условие_истинно)

Пример:

                int i = 1;
                do{
                    Console.WriteLine(i);
                    i--
                }while(i > 0)

do while выполниться хотя бы один раз

FOREACH

        foreach(объявление_переменной in название_массива){
            тело цикла
        }

Пример:

        int[] array = new int[5] {1,2,3,4,5}
        foreach(int el in array){
            Console.WriteLine("EL:" + el);
        }

Цикл FOREACH удобно использовать, чтобы перебрать многомерный массив
