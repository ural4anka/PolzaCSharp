Рандомные числа

Для генерации случайных чисел в программаз, написанных на С#, предназначен класс Random

        Random ran = new Random();

Сначало создаем объект типа Random, потом вызываем его метод Next и получаем случайное число 

        int a;
        a = rand.Newxt(100);
        a = rand.Newxt(0,11);
