Указатель "this" в классах

Использование указателя this:

        public int number;

        public int Print(int number){
            this.number = number;
        }

        здесь this обозначает, что поле number этого класса принимает параметр number от метода Print()

Указатель this не работает с модификатором static, так как поле или метод принадлежать к классу в целом
