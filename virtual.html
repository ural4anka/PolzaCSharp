Виртуальные методы

Нужны, чтобы менять метод в наследуемом классе

        class Robot {
            ...
            public virtual void PrintValues(){
                Console.WriteLine(this.name + "weight" + this.weight + "...");
            }
        }
        class Killer : Robot {
            ...
            public override void PrintValues(){
                base.PrintValues();
                Console.WriteLine("health" + this.health + "...");
            }
        }
        class Main(){
            Robot robot = new Robot("Joe", 200);
            robot.PrintValues();
            Killer killer = new Killer("Joe", 200);
            killer.PrintValues();
        }

Примечание: использовать вартуальный метод, чтобы добавлять действия в метод, а не для того, чтобы переписывать заного метод - это неправильно. Лучше тогда создать новый метод, чем переписывать имеющиийся
